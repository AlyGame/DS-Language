# Installation

It's mandatory to use C/CPP file.

### For Linux / macOS:

For install:

```bash
sudo make install
```

For uninstall:

```bash
sudo make remove
```
For run:

```bash
sudo apt install make
```

```bash
make test
```

```bash
./test
```


### For Windows:

Run the `install.ps1` for install.

Select option `(1-3)`:
* 1 : Install
* 2 : Uninstall
* 3 : Exit
